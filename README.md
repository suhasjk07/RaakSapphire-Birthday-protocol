## ğŸ› Troubleshooting

### Common Issues

#### Buttons Not Working

**Solution:**
- Open browser console (F12)
- Check for JavaScript errors
- Ensure JavaScript is enabled
- Try hard refresh (Ctrl+F5)

#### Photos Not Uploading

**Solution:**
- Check file size (< 5MB recommended)
- Use supported formats (JPG, PNG, GIF, WEBP)
- Ensure browser allows file access
- Check localStorage space

#### Notifications Not Showing

**Solution:**
- Verify employees have birthdays added
- Check notification lead time in settings
- Ensure dates are set correctly
- Check browser date/time settings

#### Google Drive Link Not Working

**Solution:**
- Verify Folder ID is correct
- Ensure folder is created in Google Drive
- Check folder permissions
- Update Folder ID in admin settings

### Browser Console Commands

Check current data:
```javascript
// View all employees
console.log(JSON.parse(localStorage.getItem('raaksapphire_employees')))

// View all gifts
console.log(JSON.parse(localStorage.getItem('raaksapphire_gifts')))

// View settings
console.log(JSON.parse(localStorage.getItem('raaksapphire_settings')))
```

Clear all data (reset):
```javascript
localStorage.clear()
location.reload()
```
